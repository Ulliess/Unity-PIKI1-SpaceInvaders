# 🚀 Space Invaders Co-op

Кооперативный мультиплеерный friendslop по мотивам Space Invaders.

## Описание

Два игрока управляют космическими кораблями в нижней трети экрана. Инопланетяне движутся сверху вниз. Корабли стреляют автоматически вверх. Задача — выжить и набрать очки.

**Визуальный стиль:** 2D минималистичный геометрический (вдохновлён xdxf).

## Core Gameplay

- **2 корабля** в нижней трети экрана, управление в двух плоскостях
- **Автоматическая стрельба** строго вверх
- **Столкновения** кораблей как твёрдых тел (можно "подставить" второго)
- **5 типов инопланетян** — разные размеры, формы, скорости
- **3 формы кораблей:**
  - 🟥 Прямоугольник — быстрое, слабое оружие  
  - 🔵 Круг — медленное, сильное оружие  
  - 🔺 Треугольник — среднее по силе/скорости, AoE эффект

## Быстрый старт (для новых участников)

### Что нужно
- [Unity Hub](https://unity.com/download) с установленным **Unity 6** (версия `6000.3.11f1`)
- [Git](https://git-scm.com/downloads)

### Шаги

1. **Клонируй репозиторий:**
   ```bash
   git clone https://github.com/Ulliess/Unity-PIKI1-SpaceInvaders.git
   ```

2. **Открой проект в Unity Hub:**
   - Открой Unity Hub → вкладка **Projects**
   - Нажми **Add** (кнопка сверху, рядом с "New project")
   - Выбери папку `Unity-PIKI1-SpaceInvaders/project`
   - Проект появится в списке — кликни на него, чтобы открыть

3. **Подожди первую загрузку:**
   Unity создаст папку `Library/` (кэш) — это занимает 2-5 минут при первом открытии. Это нормально.

### Правила работы с Git
- **Перед работой** всегда делай `git pull`
- **Коммить часто**, с понятными сообщениями
- Папки `Library/`, `Temp/`, `Logs/` — **НЕ коммитятся** (уже в .gitignore)
- ⚠️ `.meta` файлы **ОБЯЗАТЕЛЬНО коммитить** — без них Unity потеряет связи между объектами
